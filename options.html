Incognito & private windows

Time spent in incognito windows is tracked separately and kept only in memory (chrome.storage.session) — it is never written to disk, and it's cleared out as soon as your last private window closes, even if regular Chrome windows stay open. To let this extension see incognito tabs at all, Chrome requires you to explicitly opt in:

  1. Open chrome://extensions
  2. Find Time Tracker and click Details
  3. Enable Allow in Incognito

Font

The interface is styled with Martian Mono Nerd Font. Install it locally for the full look — the UI falls back to plain Martian Mono or a system monospace font otherwise, so it stays readable either way.

Reset
